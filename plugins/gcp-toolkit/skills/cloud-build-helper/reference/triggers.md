# Build Triggers

## Trigger Types
- GitHub/GitLab/Bitbucket repository triggers
- Cloud Source Repositories triggers
- Manual triggers
- Pub/Sub triggers

## Trigger Configuration
- Branch patterns (regex)
- Tag patterns
- File path filters (include/exclude)
- Substitution variables

## Best Practices
- Use branch patterns for environment-specific builds
- Implement file path filters to avoid unnecessary builds
- Use substitution variables for configuration
- Set up notifications for build status

# Azure Pricing Models

## Pay-As-You-Go
Standard pricing with no upfront commitment. Billed monthly based on usage.

## Reserved Instances
1-year or 3-year commitments with up to 72% savings. Best for stable, predictable workloads.

## Spot VMs
Up to 90% savings for interruptible workloads. Azure can evict with 30-second notice.

## Azure Hybrid Benefit
Use existing Windows Server and SQL Server licenses on Azure for significant savings.

## Dev/Test Pricing
Discounted rates for development and testing environments with Visual Studio subscriptions.

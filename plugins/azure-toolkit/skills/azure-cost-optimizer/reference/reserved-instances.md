# Reserved Instances

## Benefits
- Up to 72% savings vs pay-as-you-go
- 1-year or 3-year terms
- Flexible sizing and region exchange

## When to Use
- Stable, predictable workloads
- Production environments
- Long-running services

## Recommendations
- Analyze usage patterns for 30+ days
- Start with 1-year commitments
- Use Azure Advisor recommendations
- Consider VM series flexibility
